# TANGO-North-RFs
Receiver Function Scripts and Workflows used to produce the manuscript "The Anatomy of a Modern Cordillera Style Mountain System:  Northern Chile and Argentina", submitted to Journal of Geophysical Research; Solid Earth

preprint doi: https://doi.org/10.22541/essoar.15002927/v1
Initial Submission: April 4, 2026
Revisions Submitted: June 12, 2026

This generalized receiver function workflow follows the outline provided in the article "On the Viability of Using Autonomous Three‐Component Nodal Geophones to Calculate Teleseismic 
Ps Receiver Functions with an Application to Old Faithful, Yellowstone" by Dr. Kevin Ward and Dr. Fan-Chi Lin. 

Using the archived public data from EarthScope seismic data center repositories, users 
should be able to replicate the results contained in our article, as well collect those data into the presented figures. 

Additional dependencies include SAC, iterdecon (a fortran package which can be found on http://eqseis.geosc.psu.edu/cammon/HTML/RftnDocs/thecodes01.html), and shapefiles of geographic data which were 
digitized from articles cited in the text.

As the primary author, and developer for these scripts, I encourage these files be snipped and copied for other users in their own research. Please cite the accompanying article however. Also, as a point of note, as 
the main article that this software contributes to is published by the time this repository is public, these scripts will not receive any updates. Please be aware these were developed using Python 3.12.

Happy analyzing,
